# Dawn Sacffold —— SpringBoot脚手架

## 项目模块

### 1. 基础功能模块
| 模块            | 说明                                    | 后端                                                     | 前端                               |
| ------------- | ------------------------------------- | ------------------------------------------------------ | -------------------------------- |
| ✅ 登录认证        | 使用 Spring Security + JWT，支持登录/刷新Token | Spring Security 认证过滤器、JWT Token 生成与解析、Refresh Token 机制 | 登录页、错误提示、Token 持久化（localStorage） |
| ✅ 注册 / 第三方登录  | 支持邮箱注册 + 第三方（GitHub、LinuxDo、微信）       | OAuth2 接入、用户绑定逻辑                                       | 注册页、第三方登录按钮                      |
| ✅ 权限控制（RBAC）  | 按角色/权限控制资源访问                          | 用户-角色-权限 表设计 + 权限注解 + Security 拦截                      | 路由守卫、菜单权限渲染                      |
| ✅ 全局异常 & 统一响应 | 标准化 API 响应                            | `GlobalExceptionHandler` + 统一返回体                       | 前端 axios 拦截器统一处理错误               |
| ✅ 跨域 & 安全配置   | 允许前端跨域访问                              | CORS 配置                                                | —                                |


### 2. 系统管理模块
| 模块             | 功能点                        | 备注             |
| -------------- | -------------------------- | -------------- |
| 🧑 用户管理        | 用户列表、增删改查、状态启用禁用、重置密码      | 支持分页、模糊搜索      |
| 🧑‍🤝‍🧑 角色管理  | 角色的增删改查、分配权限               | 与菜单/权限联动       |
| 🔐 权限/菜单管理     | 菜单树结构、接口权限绑定、动态路由          | 适配前端 HeroUI 菜单 |
| ⚙️ 系统设置        | 站点名称、Logo、主题色、API Key、系统参数 | 可拓展成配置中心       |
| 📝 操作日志        | 登录日志、操作日志、异常日志             | 可选接入 AOP 自动记录  |
| 🌐 国际化支持（i18n） | 多语言                        | React i18next  |


### 3. 业务通用模块
| 模块          | 功能点                 | 说明                           |
| ----------- | ------------------- | ---------------------------- |
| 📊 通用表格组件   | 分页 + 搜索 + 多选 + 批量操作 | React 组件复用                   |
| 🧾 通用表单组件   | 动态表单渲染              | 后端返回 JSON，前端自动生成             |
| 🖼️ 文件/图片上传 | 支持图片/文档上传，返回 URL    | 支持本地 + OSS                   |
| 📨 消息通知     | 系统消息、公告通知           | WebSocket 可选                 |
| 🕹️ 定时任务管理  | Cron 表达式任务，启动/暂停    | 使用 `spring-task` 或 `xxl-job` |
| 📂 数据字典     | 例如：状态枚举、性别、类型       | 后端统一管理，前端自动渲染下拉              |
| 📊 数据导入/导出  | Excel/CSV           | 可用于批量导入用户/菜单等                |
| 💸 支付模块（预留）   | 如果 AI 需要按次收费               | 预留支付接口或充值功能     |

### 4. AI扩展模块
| 模块            | 功能点                        | 说明              |
| ------------- | -------------------------- | --------------- |
| 🧠 AI Key 管理  | 多 Key 轮询，状态监控              | 数据库管理 API Key   |
| ✨ AI 调用日志     | 调用记录、消耗额度统计                | 便于审计            |
| 💬 AI 工具      | 接入通用 LLM（如文心、OpenAI、智谱、阿里） | 封装统一的调用 Service |
